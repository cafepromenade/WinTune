# Open in App mode · 用應用程式模式開

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `br.chrome.app-mode` |
| **Module · 模組** | Browser Control · 瀏覽器控制 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | App mode · 應用程式 |

## English
Launch a site as a standalone app window with no tabs or toolbar.

## 粵語
將網站當成獨立應用程式視窗開，冇分頁同工具列。
