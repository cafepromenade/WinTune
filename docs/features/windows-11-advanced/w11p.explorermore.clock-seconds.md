# Show seconds in the system clock · 系統時鐘顯示秒數

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.explorermore.clock-seconds` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Display seconds on the taskbar clock (costs a little extra battery/CPU).

## 粵語
喺工作列時鐘度顯示秒數（會多用少少電/CPU）。
