# Disable USB selective suspend · 停用 USB 選擇性暫停

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.usb-selective-suspend` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Disable · 停用 |

## English
Stop Windows from powering down USB devices on the active power plan.

## 粵語
唔俾 Windows 喺目前電源計劃熄 USB 裝置電。
