# Filter Keys · 篩選鍵

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.inputintl.filter-keys` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |

## English
Ignore brief or repeated keystrokes and slow the repeat rate.

## 粵語
忽略短暫或重複嘅按鍵，並減慢重複速度。
