# Crash dump type · 當機傾印類型

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.crash-dump` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Choice |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |

## English
Set the kind of memory dump Windows writes on a blue screen; minidump is smallest.

## 粵語
設 Windows 藍畫面時寫嘅記憶體傾印類型；minidump 最細。
