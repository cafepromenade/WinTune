# Restore classic volume mixer · 還原傳統音量混音器

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.explorermore.classic-volume-mixer` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Note: Win11 routes the speaker icon to the modern Settings mixer; this MTCUVC flag restores the classic SndVol flyout where the build still supports it.

## 粵語
注意：Win11 將喇叭圖示帶去新版設定混音器；呢個 MTCUVC 旗標喺仲支援嘅組建度還原傳統 SndVol 飛出視窗。
