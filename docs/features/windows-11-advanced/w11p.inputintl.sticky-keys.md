# Sticky Keys · 相黏鍵

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.inputintl.sticky-keys` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |

## English
Let modifier keys (Shift, Ctrl, Alt) stay active without holding them.

## 粵語
等修飾鍵（Shift、Ctrl、Alt）唔使撳住都保持作用。
