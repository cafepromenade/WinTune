# Virtual desktops on all monitors · 所有螢幕顯示虛擬桌面

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.storagenotif.vd-all-monitors` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |

| **Destructive · 具破壞性** | No · 唔係 |

When switching virtual desktops, switch on all monitors instead of just one.

切換虛擬桌面嗰陣同時喺所有螢幕切換而唔係淨係一個。
