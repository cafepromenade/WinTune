# GPU TDR delay · GPU TDR 延遲

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.tdr-delay` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Reboot |

## English
Increase the GPU timeout detection and recovery delay to 8 seconds to reduce driver-reset crashes.

## 粵語
將 GPU 逾時偵測復原（TDR）延遲增至 8 秒，減少驅動重置當機。
