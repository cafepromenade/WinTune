# Disable the lock screen · 停用鎖定畫面

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.explorermore.disable-lockscreen` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Reboot |

## English
Note: skip the lock screen so logon goes straight to the password prompt. Requires the Personalization policy key (no-op on some Home builds).

## 粵語
注意：跳過鎖定畫面，登入直接去密碼提示。要 Personalization 原則機碼（部分家用版唔生效）。
