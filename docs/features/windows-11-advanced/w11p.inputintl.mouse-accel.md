# Disable mouse acceleration · 停用滑鼠加速

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.inputintl.mouse-accel` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |

## English
Turn off Enhance Pointer Precision so mouse movement is 1:1.

## 粵語
關閉「增強指標精確度」，令滑鼠移動係 1:1。
