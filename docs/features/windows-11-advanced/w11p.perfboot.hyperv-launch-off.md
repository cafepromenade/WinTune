# Hypervisor launch off (bcdedit) · 關閉虛擬器啟動（bcdedit）

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.hyperv-launch-off` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Reboot |
| **Action · 動作** | Set Off · 設 Off |

## English
Use bcdedit to set hypervisorlaunchtype to Off; can improve game performance. Reboot required.

## 粵語
用 bcdedit 將 hypervisorlaunchtype 設為 Off；可提升遊戲效能。要重啟先生效。
