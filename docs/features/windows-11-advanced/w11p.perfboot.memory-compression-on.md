# Enable memory compression · 啟用記憶體壓縮

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.memory-compression-on` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Reboot |
| **Action · 動作** | Enable · 啟用 |

## English
Turn on MMAgent memory compression to fit more data in RAM and reduce paging.

## 粵語
開啟 MMAgent 記憶體壓縮，喺 RAM 裝多啲資料、減少分頁。
