# Disable memory compression · 停用記憶體壓縮

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.memory-compression-off` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Reboot |
| **Action · 動作** | Disable · 停用 |

## English
Turn off MMAgent memory compression; can reduce CPU use on systems with plenty of RAM.

## 粵語
關閉 MMAgent 記憶體壓縮；RAM 充足嘅機可減少 CPU 用量。
