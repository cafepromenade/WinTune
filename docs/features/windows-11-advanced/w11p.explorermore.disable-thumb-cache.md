# Disable thumbnail caching · 停用縮圖快取

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.explorermore.disable-thumb-cache` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Stop Explorer from caching thumbnails in thumbcache_*.db; thumbnails regenerate each time.

## 粵語
唔再喺 thumbcache_*.db 度快取縮圖，每次重新產生縮圖。
