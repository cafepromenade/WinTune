# Disable startup app delay · 停用啟動程式延遲

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.startup-delay` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Remove the artificial delay before startup apps launch after sign-in.

## 粵語
移除登入後啟動程式開始前嘅人為延遲。
