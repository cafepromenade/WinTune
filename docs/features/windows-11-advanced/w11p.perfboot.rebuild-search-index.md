# Rebuild search index · 重建搜尋索引

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.rebuild-search-index` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Reboot |

## English
Flag the Windows Search index for a full rebuild on next service start.

## 粵語
標記 Windows 搜尋索引，喺服務下次啟動時完整重建。
