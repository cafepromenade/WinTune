# Always show all tray icons (legacy) · 永遠顯示所有系統匣圖示（舊版）

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.explorermore.tray-all-icons` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Note: Win11 has no single registry switch; use Settings > Personalization > Taskbar > Other system tray icons. This flips the legacy EnableAutoTray flag (affects older shells).

## 粵語
注意：Win11 冇單一登錄機碼，要去設定 > 個人化 > 工作列 > 其他系統匣圖示。呢個掣係改舊版 EnableAutoTray 旗標（影響舊版殼層）。
