# Disable typing insights · 停用輸入分析

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `winaero.privacy-network.disable-typing-insights` |
| **Module · 模組** | Winaero Tweaks · Winaero 進階調校 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |

## English
Sets InsightsEnabled to 0 so Windows stops collecting typing insights from your text suggestions and autocorrect.

## 粵語
將 InsightsEnabled 設為 0，令 Windows 唔再收集你打字嘅文字建議同自動更正分析。
