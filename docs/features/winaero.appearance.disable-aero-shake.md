# Disable window shake (Aero Shake) to minimize · 停用搖晃視窗（Aero Shake）最小化

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `winaero.appearance.disable-aero-shake` |
| **Module · 模組** | Winaero Tweaks · Winaero 進階調校 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Stops Windows from minimizing all other windows when you grab and shake a window's title bar. Sets DisallowShaking=1; default behavior returns when the value is 0/removed.

## 粵語
當你抓住搖晃某視窗標題列時，唔再最小化其他所有視窗。設定 DisallowShaking=1；值為 0／移除時恢復預設行為。
