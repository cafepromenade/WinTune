# Disable feedback request frequency · 停用意見反映要求頻率

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `winaero.privacy-network.disable-feedback-frequency` |
| **Module · 模組** | Winaero Tweaks · Winaero 進階調校 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |

## English
Sets NumberOfSIUFInPeriod to 0 so Windows never asks for feedback through the SIUF survey system.

## 粵語
將 NumberOfSIUFInPeriod 設為 0，令 Windows 唔會再透過 SIUF 問卷系統要求意見反映。
