# Reset DNS to DHCP · DNS 還原做 DHCP

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `net.ipdns.reset-dhcp` |
| **Module · 模組** | Network Pro · 網絡進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Reset · 還原 |

Clear manual DNS servers so the adapter gets DNS from DHCP again.

## 粵語
清走手動設定嘅 DNS 伺服器，等網絡卡重新由 DHCP 攞返 DNS。
