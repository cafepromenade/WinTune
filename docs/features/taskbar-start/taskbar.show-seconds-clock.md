# Show seconds in system clock · 系統時鐘顯示秒數

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `taskbar.show-seconds-clock` |
| **Module · 模組** | Taskbar & Start · 工作列與開始功能表 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |

喺工作列時鐘顯示秒數（會用多啲電）。
