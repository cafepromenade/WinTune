# Block a network · 封鎖一個網絡

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `net.wifi.block-network` |
| **Module · 模組** | Network Pro · 網絡進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Block · 封鎖 |

## English
Add a deny filter to block a specific SSID. Replace SSID with the network name.

## 粵語
加一個拒絕篩選去封鎖指定 SSID。將 SSID 換成網絡名啦。
