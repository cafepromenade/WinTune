# Disable power throttling · 停用電源節流

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `performance.power-throttling-off` |
| **Module · 模組** | Performance & Power · 效能與電源 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |

## English
Stop Windows from throttling background apps to save power.

## 粵語
唔再為咗慳電而節流背景程式。
