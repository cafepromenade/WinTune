# Show window contents while dragging · 拖曳時顯示視窗內容

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `appearance.drag-full-windows` |
| **Module · 模組** | Appearance & Personalisation · 外觀與個人化 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | SignOut |

## English
Show the whole window instead of an outline while dragging.

## 粵語
拖曳視窗嗰陣顯示成個視窗內容，而唔係淨係框線。
