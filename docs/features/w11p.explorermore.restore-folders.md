# Restore folder windows at logon · 登入時還原資料夾視窗

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.explorermore.restore-folders` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Reopen the Explorer windows that were open when you last signed out.

## 粵語
登入返時，重新開返你上次登出前開住嘅總管視窗。
