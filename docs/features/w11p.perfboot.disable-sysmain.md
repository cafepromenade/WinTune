# Disable SysMain (Superfetch) · 停用 SysMain（Superfetch）

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.disable-sysmain` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | Yes · 係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Disable · 停用 |

## English
Set the SysMain service to disabled; can help on SSDs by cutting background disk activity.

## 粵語
將 SysMain 服務設為停用；SSD 機可減少背景磁碟活動。
