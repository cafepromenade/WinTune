## English
Suspend BitLocker protection on C: for exactly one reboot, then it auto-resumes. Useful before firmware/TPM changes.

## 粵語
暫停 C: 嘅 BitLocker 保護，淨係跳過一次重開機，之後自動恢復。改韌體或 TPM 之前好用。
