# Storage reliability counters · 儲存可靠性計數器

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `maint.disk-reliability` |
| **Module · 模組** | Maintenance & Diagnostics · 維護與診斷 |

## English
Read SMART-style reliability counters (temperature, wear, read/write errors) for each disk.

## 粵語
讀取每個磁碟嘅可靠性計數器（溫度、磨損、讀寫錯誤），類似 SMART。
