# Enable BitLocker on a drive · 為磁碟啟用 BitLocker

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `vault.bitlocker.enable-drive` |
| **Module · 模組** | Encryption & Vault · 加密與保險庫 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Enable D: · 啟用 D: |

## English
Turn on BitLocker encryption for drive D: with a recovery password protector, encrypting used space only. Change the letter as needed.

## 粵語
為 D: 磁碟開啟 BitLocker 加密，用還原密碼保護器，只加密已用空間。需要時自行改字母。
