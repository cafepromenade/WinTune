# Launch in safe diagnostic mode · 用安全診斷模式啟動

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `br.chrome.safe-mode` |
| **Module · 模組** | Browser Control · 瀏覽器控制 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Safe mode · 安全模式 |

## English
Start Chrome with extensions disabled and a clean temporary profile for diagnosis.

## 粵語
停用擴充功能再用一個乾淨臨時設定檔嚟啟動 Chrome，方便診斷。
