# Renew IP address · 更新 IP 位址

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `network.renew-ip` |
| **Module · 模組** | Network & Internet · 網絡與互聯網 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |

Requests a fresh DHCP lease for all adapters.

向 DHCP 重新攞過一個 IP 租約畀所有網絡卡。
