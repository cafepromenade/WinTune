# Enable Num Lock at startup · 開機時開啟 Num Lock

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `winaero.boot-logon.numlock-at-startup` |
| **Module · 模組** | Winaero Tweaks · Winaero 進階調校 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | SignOut |

## English
Turns Num Lock on automatically at the sign-in screen before anyone logs in.

## 粵語
喺登入畫面、任何人登入之前自動開啟 Num Lock。
