# Disable telemetry autologger (DiagTrack) · 停用遙測自動記錄器 (DiagTrack)

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `winaero.privacy-network.disable-telemetry-autologger` |
| **Module · 模組** | Winaero Tweaks · Winaero 進階調校 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Reboot |

## English
Sets the AutoLogger-Diagtrack-Listener ETW session Start to 0 so the DiagTrack telemetry trace does not run at boot.

## 粵語
將 AutoLogger-Diagtrack-Listener 嘅 Start 設為 0，令 DiagTrack 遙測追蹤開機時唔運行。
