# Remove "Cast to Device" from context menu · context menu 移除「投放到裝置」

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `winaero.context-menu.remove-cast-to-device` |
| **Module · 模組** | Winaero Tweaks · Winaero 進階調校 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | SignOut |

## English
Hides the "Cast to Device" (Play To) entry from the right-click menu of media files by blocking its shell extension. Disable to restore it.

## 粵語
封鎖個 shell extension，喺媒體檔嘅右掣 menu 度收埋「投放到裝置」(Play To)。撳返 disable 就會還原。
