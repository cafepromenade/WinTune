| Field · 欄位 | Value · 值 |

## English
Display drive letters before the drive label in This PC (e.g. (C:) Local Disk).

## 粵語
喺「本機」將磁碟機代號顯示喺名稱之前（例如 (C:) 本機磁碟）。
