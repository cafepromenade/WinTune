# Remove "Restore previous versions" from context menu · context menu 移除「還原舊版本」

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `winaero.context-menu.remove-restore-previous-versions` |
| **Module · 模組** | Winaero Tweaks · Winaero 進階調校 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | SignOut |

## English
Hides the "Restore previous versions" context menu entry and the Previous Versions properties tab by blocking its shell extension. Disable to restore it.

## 粵語
封鎖個 shell extension，收埋「還原舊版本」嘅右掣項目同埋內容頁嘅「之前版本」分頁。撳返 disable 就會還原。
