# Show taskbar on all displays · 所有螢幕顯示工作列

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `taskbar.multi-monitor-all` |
| **Module · 模組** | Taskbar & Start · 工作列與開始功能表 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Show the taskbar on every connected monitor.
