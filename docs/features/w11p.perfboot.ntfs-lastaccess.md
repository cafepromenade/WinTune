# Disable NTFS last-access · 停用 NTFS 最後存取

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.ntfs-lastaccess` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Disable · 停用 |

## English
Stop NTFS from updating the last-access timestamp on every file read to cut disk overhead.

## 粵語
唔俾 NTFS 每次讀檔都更新最後存取時間，減少磁碟負擔。
