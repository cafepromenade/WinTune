# Launch with GPU disabled · 停用 GPU 啟動

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `br.chrome.disable-gpu` |
| **Module · 模組** | Browser Control · 瀏覽器控制 |
| **Type · 種類** | Action |

Start Chrome with hardware GPU acceleration turned off to fix rendering glitches.

關閉硬件 GPU 加速嚟啟動 Chrome，用嚟修正畫面顯示問題。
