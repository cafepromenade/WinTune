# Disable reserved storage · 停用保留儲存空間

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.free-reserved` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Disable · 停用 |

## English
Free the ~7GB Windows reserves for updates by disabling reserved storage.

## 粵語
釋放 Windows 為更新預留嘅約 7GB 空間，停用保留儲存。
