# Maximise icon cache size · 加大圖示快取容量

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.explorermore.max-icon-cache` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Raise the icon cache limit to 4096 KB so large icon sets stay cached and load faster.

## 粵語
將圖示快取上限調到 4096 KB，等大量圖示留喺快取度，載入快啲。
