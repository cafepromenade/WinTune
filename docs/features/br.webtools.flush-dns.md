# Flush DNS cache · 清除 DNS 快取

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `br.webtools.flush-dns` |
| **Module · 模組** | Browser Control · 瀏覽器控制 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Flush · 清除 |

## English
Clear the system DNS resolver cache so browsers re-resolve hostnames.

## 粵語
清除系統 DNS 解析快取，等瀏覽器重新解析網域名稱。
