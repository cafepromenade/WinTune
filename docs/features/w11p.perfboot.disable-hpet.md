# Disable HPET timer · 停用 HPET 計時器

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.disable-hpet` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | Yes · 係 |
| **Restart · 重啟** | Reboot |
| **Action · 動作** | Disable · 停用 |

## English
Disable the High Precision Event Timer device; some systems see lower latency. Re-enable in Device Manager if unstable.

## 粵語
停用高精度事件計時器（HPET）裝置；部分機可降低延遲。如唔穩定喺裝置管理員重新啟用。
