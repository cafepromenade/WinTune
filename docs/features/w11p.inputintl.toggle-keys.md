# Toggle Keys · 切換鍵

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.inputintl.toggle-keys` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |

## English
Play a tone when Caps Lock, Num Lock or Scroll Lock is pressed.

## 粵語
撳 Caps Lock、Num Lock 或 Scroll Lock 時播放提示音。
