# Set default inbound to block · 設定預設入站為封鎖

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `net.firewall.default-inbound-block` |
| **Module · 模組** | Network Pro · 網絡進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Apply · 套用 |

## English
Set the default inbound action to block on all profiles so unmatched inbound traffic is dropped.

## 粵語
將所有設定檔嘅預設入站動作設為封鎖，未配對嘅入站流量會被丟棄。
