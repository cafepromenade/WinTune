| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `system.long-paths` |
| **Module · 模組** | System & Boot · 系統與開機 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | Yes · 需要 |
