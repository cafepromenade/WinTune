# Allow ping (ICMP echo) · 允許 Ping（ICMP）

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `net.firewall.allow-ping` |
| **Module · 模組** | Network Pro · 網絡進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Allow · 允許 |

## English
Add an inbound rule allowing ICMPv4 echo requests so the PC responds to ping.

## 粵語
加入站規則允許 ICMPv4 回應請求，等部電腦識回應 ping。
