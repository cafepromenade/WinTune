# Disable Aero Shake to minimise · 停用 Aero 搖晃最小化

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.explorermore.disable-shake` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Stop other windows from minimising when you shake the active window's title bar.

## 粵語
搖動使用中視窗嘅標題列時，唔好再最小化其他視窗。
