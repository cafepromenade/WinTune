# Show system uptime · 顯示系統運行時間

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `dev.envports.uptime` |
| **Module · 模組** | Developer & Terminal · 開發與終端機 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Show · 顯示 |

## English
Show how long since the last boot using the last boot-up time.

## 粵語
用上次開機時間計算系統已運行幾耐。
