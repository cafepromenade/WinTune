# Reset DNS to automatic · DNS 還原自動

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `network.dns-auto` |
| **Module · 模組** | Network & Internet · 網絡與互聯網 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Reset · 還原 |

## English
Clears manual DNS so the adapter gets DNS from DHCP again.

## 粵語
清除手動 DNS，等網絡卡返去用 DHCP 派嘅 DNS。
