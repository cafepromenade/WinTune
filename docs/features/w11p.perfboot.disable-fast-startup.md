# Disable Fast Startup · 停用快速啟動

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.perfboot.disable-fast-startup` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Reboot |
| **Action · 動作** | Disable · 停用 |

## English
Turn off hybrid Fast Startup so shutdown fully clears state; fixes some boot and driver issues.

## 粵語
關閉混合快速啟動，令關機完整清除狀態；解決部分開機同驅動問題。
