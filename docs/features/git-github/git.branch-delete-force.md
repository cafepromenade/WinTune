| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `git.branch-delete-force` |

Force-delete the branch 'feature' even if it is not merged. This loses unmerged work.

## 粵語
就算 'feature' 未合併都強制刪除，會整唔見未合併嘅嘢。
