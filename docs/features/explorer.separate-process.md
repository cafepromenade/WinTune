# Launch folder windows in a separate process · 資料夾視窗用獨立程序開啟

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `explorer.separate-process` |
| **Module · 模組** | File Explorer · 檔案總管 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

Open each File Explorer window in its own process for stability.
