## English
Telnet is the classic way to test a raw TCP port, but the client is off by default. Use Test-NetConnection instead, shown here against port 80.

## 粵語
Telnet 係測試原始 TCP 埠嘅經典方法，但客戶端預設冇裝。建議改用 Test-NetConnection，呢度示範測 80 埠呀。
