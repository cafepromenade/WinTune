# Turn firewall off (all profiles) · 關閉防火牆（所有設定檔）

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `net.firewall.turn-all-off` |

| **Type · 種類** | Action |

| **Destructive · 具破壞性** | Yes · 係 |
