# Force recovery on next boot · 下次開機逼出還原畫面

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `vault.bitlocker.force-recovery` |
| **Module · 模組** | Encryption & Vault · 加密與保險庫 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | Yes · 係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Force recovery · 逼出還原 |

## English
Force C: into BitLocker recovery so the very next boot demands the 48-digit recovery key (useful before firmware/TPM changes). DANGEROUS: you cannot boot without the recovery key. Change the letter as needed.

## 粵語
強制 C: 進入 BitLocker 還原狀態，下次開機就要打 48 位還原密碼先開到機（改韌體或 TPM 之前好用）。危險：冇還原密碼就開唔到機。需要時自行改字母。
