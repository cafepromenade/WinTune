# Enable auto-unlock · 啟用自動解鎖

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `vault.bitlocker.enable-autounlock` |
| **Module · 模組** | Encryption & Vault · 加密與保險庫 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Enable D: · 啟用 D: |

## English
Enable automatic unlocking of fixed data drive D: when the system drive is unlocked (system drive must already be BitLocker-protected). Change the letter as needed.

## 粵語
啟用固定資料磁碟 D: 嘅自動解鎖，系統碟解鎖時佢就跟住解（系統碟要已經有 BitLocker 保護）。需要時自行改字母。
