# Repair a damaged drive (repair-bde) · 修復爛咗嘅磁碟 (repair-bde)

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `vault.bitlocker.repair-bde` |
| **Module · 模組** | Encryption & Vault · 加密與保險庫 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | Yes · 係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Repair · 修復 |

## English
Reconstruct and decrypt salvageable data from a corrupted BitLocker volume to a SEPARATE empty target, using the 48-digit recovery password. The target is overwritten, so it must be a different, empty drive/image. Edit the input volume, output target and recovery password first.

## 粵語
用 48 位還原密碼，將損壞嘅 BitLocker 磁碟區可救嘅資料重建並解密到另一個空白目標。目標會被覆寫，所以一定要係另一隻空白磁碟／映像。先改輸入磁碟區、輸出目標同還原密碼。
