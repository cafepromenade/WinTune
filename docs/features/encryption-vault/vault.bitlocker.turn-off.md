# Turn off BitLocker (decrypt) · 關閉 BitLocker（解密）

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `vault.bitlocker.turn-off` |
| **Module · 模組** | Encryption & Vault · 加密與保險庫 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | Yes · 係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Turn off D: · 關 D: |

## English
Fully decrypt drive D: and turn BitLocker off. This can take a long time and removes encryption protection. Change the letter as needed.

## 粵語
完全解密 D: 磁碟並關閉 BitLocker。可能要好耐，亦會移除加密保護。需要時自行改字母。
