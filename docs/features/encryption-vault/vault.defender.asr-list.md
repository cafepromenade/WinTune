# List ASR rules (with names) · 列出 ASR 規則（連名）

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `vault.defender.asr-list` |
| **Module · 模組** | Encryption & Vault · 加密與保險庫 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | List · 列出 |

## English
List every configured Attack Surface Reduction rule with its human-readable name and current action (Enabled / AuditMode / Disabled / Warn). Names are resolved at runtime from the GUID, so unknown rules still show their GUID.

## 粵語
列出每條已設定嘅攻擊面收窄（ASR）規則，連人類可讀名稱同目前動作（啟用／稽核／停用／警告）。名稱喺執行時由 GUID 解析，未知規則都會顯示 GUID。
