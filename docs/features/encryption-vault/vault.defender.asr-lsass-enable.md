# ASR: block LSASS theft (Enabled) · ASR：封鎖 LSASS 竊取（啟用）

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `vault.defender.asr-lsass-enable` |
| **Module · 模組** | Encryption & Vault · 加密與保險庫 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Enable · 啟用 |

## English
Enable the Attack Surface Reduction rule that blocks credential stealing from the Windows local security authority (lsass.exe). Set to Enabled (block).

## 粵語
啟用「封鎖由 Windows 本機安全性授權（lsass.exe）竊取憑證」嘅攻擊面收窄規則，設為「啟用」（封鎖）。
