# Forget a network · 忘記一個網絡

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `net.wifi.forget-network` |
| **Module · 模組** | Network Pro · 網絡進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | Yes · 係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Forget · 忘記 |

## English
Delete a saved Wi-Fi profile so Windows forgets it. Replace PROFILE with the profile name.

## 粵語
刪除已儲存嘅 Wi-Fi 設定檔，等 Windows 唔記得佢。將 PROFILE 換成設定檔名啦。
