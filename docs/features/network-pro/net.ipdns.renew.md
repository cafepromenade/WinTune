# Renew IP address · 更新 IP 位址

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `net.ipdns.renew` |
| **Module · 模組** | Network Pro · 網絡進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Renew · 更新 |

## English
Request a fresh DHCP lease and renew the IPv4 address on all adapters.

## 粵語
重新向 DHCP 要一個新租約，更新所有網絡卡嘅 IPv4 位址。
