# Re-register DNS · 重新註冊 DNS

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `net.ipdns.registerdns` |
| **Module · 模組** | Network Pro · 網絡進階 |
| **Type · 種類** | Action |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |
| **Action · 動作** | Register · 註冊 |

## English
Refresh DHCP leases and re-register this PC's DNS names with the server.

## 粵語
刷新 DHCP 租約，並向伺服器重新註冊呢部電腦嘅 DNS 名稱。
