# Launch folder windows in a separate process · 用獨立處理序開資料夾視窗

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.explorermore.separate-process` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | Explorer |

## English
Run each Explorer window in its own process so one crash doesn't take down the rest.

## 粵語
每個總管視窗用自己嘅處理序行，一個崩潰唔會拖冚其他。
